# Shortlinks

Handler for go.cs61a.org and links.cs61a.org shortlinks.

## Local Development

Create and activate a venv:

```bash
$ python3 -m venv env
$ source env/bin/activate
```

Use `pip` to install all the dependencies:

```bash
$ pip install -r requirements.txt
```

Run the server:

```bash
$ python main.py
```
